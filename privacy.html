Privacy Policy

Effective Date: February 24, 2026

1. Data Collected

This bot may store the following Discord-related data:

  • Discord Server IDs
  • Channel IDs
  • Role IDs
  • User IDs (only when required for bot functionality)

The bot does not collect:

  • Email addresses
  • Passwords
  • Payment information
  • Private account credentials
  • Direct messages outside bot interaction

2. Purpose of Data

Stored data is used strictly to:

  • Provide reminders and event notifications
  • Manage verification systems
  • Configure server-specific settings
  • Enable role-based functionality

3. Data Storage

Data is stored securely and is not sold, rented, or shared with third parties. Only data necessary for bot functionality is retained.

4. Third-Party Services

This bot interacts with the following services:

  • Discord API
  • Google Calendar public ICS feeds

No additional third-party tracking or analytics services are used.

5. Data Removal

Server owners may request deletion of stored configuration data at any time. Data related to a server may also be removed if the bot is removed from that server.
